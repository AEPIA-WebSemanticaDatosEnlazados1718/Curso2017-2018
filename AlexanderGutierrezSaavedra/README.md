# Introducción
Promover una fuente de datos enlazados, requiere conocer con mas detalle los datos origen, desde su estructura, hasta su dominio. Teniendo claridad en estos puntos es posible diseñar la forma como se abordarán los datos y determinar si requiere o no la construcción de ontologías que permitan enmarcar los datos. promoviendo la generación de datos enlazados, se deben crear las ontologías requeridas, realizar depuración de la data y aplicar algunas reglas para finalmente obtener la información requerida de manera que puedan ser mapeadas y enlzadas a partir de las ontologías construidas. 
En este trabajo se pretende crear una red ontológica a partir de unos datos origen seleccionados en fomrato estandar.
