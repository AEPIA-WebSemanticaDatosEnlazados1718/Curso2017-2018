# Introducción
Promover una fuente de datos enlazados, requiere conocer con detalle los datos origen, desde su estructura, hasta su dominio. Teniendo claridad en estos puntos es posible diseñar la forma como se abordarán los datos y determinar si requiere o no la construcción de ontologías que permitan enmarcar los datos. Una vez se resuelven las Ontologías, se hace necesario depurar la data y aplicar algunas reglas para finalmente generar el conjunto de datos requeridos. En este trabajo se explica cómo se realizaron estos pasos con mayor detalle para dar un entendimiento más amplio de las implicaciones que se dieron a lo largo del trabajo.
